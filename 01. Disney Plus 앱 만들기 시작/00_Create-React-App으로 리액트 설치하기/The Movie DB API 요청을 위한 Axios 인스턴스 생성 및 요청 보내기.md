## Axios 인스턴스 생성 및 요청 보내기 

##### The Movie DB API 요청을 위한 Axios 인스턴스 생성 및 요청 보내기 



#### ✔ Axios? 

* Axios : 브라우저, Node.js를 위한 Promise API를 활용하는 HTTP 비동기 통신 라이브러리 
* 벡앤드와 프론트엔드 통신을 쉽게 하기 위해 Ajax와 더불어 사용한다.

npm install axios --save
